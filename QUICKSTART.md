### 创建房间

1. 点击 "🏠 创建房间"
2. 获得 6 位房间号（如：A1B2C3）
3. 点击 "📤 分享房间" 分享给好友

1. 点击 "🚪 加入房间"
2. 输入 6 位房间号
3. 点击确定
4. 加入成功，等待开始

## 🃏 游戏流程

### 1. 发牌阶段

```
每人 17 张牌，留 3 张底牌
```

### 2. 叫分阶段

```
┌──────────────────────────┐
│    请叫分                │
├──────────────────────────┤
│  🎁 底牌：???            │
├──────────────────────────┤
│  [1 分] [2 分] [3 分]    │
│  [不要]                  │
└──────────────────────────┘
```

- **1/2/3 分** - 叫分，分数越高奖励越多
- **不要** - 放弃叫分
- 叫分最高者成为地主，获得 3 张底牌

### 3. 出牌阶段

```
┌──────────────────────────┐
│  玩家 1  🂠🂠🂠🂠🂠       │
│  玩家 2  🂠🂠🂠🂠🂠       │
├──────────────────────────┤
│  [你的牌]                │
│  🂡 🂢 🂣 🂤 🂥 ...      │
├──────────────────────────┤
│  轮到你了                │
├──────────────────────────┤
│  [不要] [出牌] [🤖托管]  │
└──────────────────────────┘
```

- **点击牌** - 选择/取消
- **出牌** - 打出选中的牌
- **不要** - 跳过本轮
- **托管** - 开启自动出牌

### 4. 游戏结束

```
┌──────────────────────────┐
│    🎉 你赢了！           │
├──────────────────────────┤
│  恭喜你获胜！            │
│  积分变化：+80           │
│  (地主，炸弹 x1)         │
├──────────────────────────┤
│  [🏠 返回大厅]           │
└──────────────────────────┘
```

## 📊 功能说明

### 积分榜

- 查看当前积分
- 查看当前段位
- 段位：黑铁 → 白银 → 黄金 → 铂金 → 钻石 → 大师 → 宗师 → 王者

### 战绩

- 总对局数
- 胜利/失败次数
- 胜率
- 地主胜利/农民胜利
- 最近 10 局记录

### 排行榜

- 全服前 10 名
- 我的排名
- 积分对比
- 对局数量

### 每日签到

- 每天可签到一次
- 获得 10 分奖励
- 自动累计积分

## 🎯 快速技巧

### 新手技巧

1. **先熟悉牌型** - 了解各种牌型大小
2. **合理叫分** - 牌好再叫地主
3. **记住牌** - 记住已出的大牌
4. **配合队友** - 农民要配合打地主

### 进阶技巧

1. **算牌** - 根据出牌推测对手牌型
2. **控牌** - 控制出牌权
3. **诱牌** - 引诱对手出炸弹
4. **忍牌** - 该忍则忍，不要硬拼

### 托管使用

- **何时使用**: 临时有事/新手练习
- **注意**: 托管 AI 较简单，建议手动玩

## ❓ 常见问题

### Q: 如何开始游戏？
A: 发送 `/game` → 创建/加入房间 → 开始游戏

### Q: 如何叫地主？
A: 发牌后选择叫分（1/2/3 分）或不要

### Q: 如何出牌？
A: 点击手牌选择 → 点击 "出牌" 按钮

### Q: 如何托管？
A: 点击 "🤖 托管" 按钮，系统自动出牌

### Q: 如何查看战绩？
A: 大厅点击 "📜 战绩" 按钮

### Q: 如何签到？
A: 大厅点击 "📅 每日签到" 按钮

### Q: 积分有什么用？
A: 提升段位，解锁成就，排行榜排名
